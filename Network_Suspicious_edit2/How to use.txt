# วิธีการใช้งาน NetDetect

โปรแกรมนี้ใช้สำหรับวิเคราะห์ไฟล์ .pcap เพื่อตรวจจับกิจกรรมที่น่าสงสัยในเครือข่าย โดยใช้ระบบเรียนรู้ค่าพื้นฐาน (Baseline)

## สิ่งที่ต้องมีก่อนติดตั้ง

- Python 3.10 ขึ้นไป
- ไลบรารีที่จำเป็น (ดูใน requirements.txt)

## การติดตั้ง

1. แนะนำให้ใช้งานใน Virtual Environment
2. ติดตั้งไลบรารีที่จำเป็นทั้งหมดด้วยคำสั่ง:
   pip install -r requirements.txt

## ขั้นตอนการใช้งาน

โปรแกรมมี 2 โหมดหลักคือ `learn` และ `detect`

### ขั้นตอนที่ 1: สร้างค่าพื้นฐานของเครือข่าย (โหมดเรียนรู้)

คุณต้อง "สอน" ให้โปรแกรมรู้จักทราฟฟิกปกติของเครือข่ายก่อน โดยใช้ไฟล์ .pcap ที่มีแต่ข้อมูลปกติ (ไม่มีการโจมตี)

**คำสั่ง:**
python Network_Suspicious.py learn --pcap path/to/normal_traffic.pcap

เมื่อรันคำสั่งนี้ โปรแกรมจะสร้างไฟล์ `network_baseline.json` ขึ้นมาในโฟลเดอร์ `result/` คุณต้องทำขั้นตอนนี้เพียงครั้งเดียว หรือเมื่อต้องการเรียนรู้ค่าพื้นฐานใหม่

### ขั้นตอนที่ 2: ตรวจจับกิจกรรมที่น่าสงสัย (โหมดตรวจจับ)

หลังจากมีไฟล์ Baseline แล้ว คุณสามารถนำโปรแกรมไปวิเคราะห์ไฟล์ .pcap ใดๆ ก็ได้

**คำสั่ง:**
python Network_Suspicious.py detect --pcap path/to/traffic_to_analyze.pcap

หากตรวจพบกิจกรรมที่น่าสงสัย โปรแกรมจะบันทึกรายงานสรุปผลไว้ที่ไฟล์ `detection_report.txt` ในโฟลเดอร์ `result/`

### การตั้งค่าเพิ่มเติม

คุณสามารถปรับความไวในการตรวจจับได้โดยแก้ไขค่า `sensitivity` ในไฟล์ `config.ini`
- ค่าน้อย (เช่น 2.5) จะทำให้ไวต่อความผิดปกติมากขึ้น (อาจเจอ False Positive)
- ค่ามาก (เช่น 3.5) จะทำให้ไวต่อความผิดปกติน้อยลง
